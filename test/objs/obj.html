OBJ Files
A 3D Object Format

OBJ is a data directory which contains examples of OBJ files, a format for storing a description of the surface of a 3D object, composed of triangles or higher degree polygons.

OBJ File Characteristics:

  • ASCII (there is also a binary version, for which an "MOD" extension is used in the file name);
  • Color;
  • 3D;
  • No compression;
  • contain definitions of 1 or more named 3D objects;

Licensing:

Related Data and Programs:

MESHLAB, examples which illustrate the use of the meshlab program, an advanced mesh processing system for automatic or user-assisted editing, cleaning, filtering, converting and rendering of large unstructured 3D triangular meshes.

MTL, a data directory which describes a file format used by OBJ to set material types.
